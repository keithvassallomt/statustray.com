# Basic Usage

## Running the Extension

If you installed the extension from the GNOME Extensions website, it should be enabled automatically. If you need to enable it manually:

1. Open the GNOME [Extensions](https://flathub.org/en/apps/org.gnome.Extensions) or [Extension Manager](https://flathub.org/en/apps/com.mattjakeman.ExtensionManager) app.
2. Find "Status Tray" in the list of installed extensions.
3. Toggle the switch to enable it.

![Enable Status Tray in Extension Manager](/extension_manager.png)

## Basic Settings

If all you want is for status icons to be displayed in GNOME, you don't need to change any settings! However, you can customise the behaviour of the extension by opening the settings.

1. Open the GNOME [Extensions](https://flathub.org/en/apps/org.gnome.Extensions) or [Extension Manager](https://flathub.org/en/apps/com.mattjakeman.ExtensionManager) app.
2. Find "Status Tray" in the list of installed extensions.
3. Click the gear icon to open the settings.

![Status Tray settings](/settings.png)

### Icon Style
You can choose between two icon styles:
- **Symbolic (monochrome)**: These icons are designed to match your system theme (light or dark) and are monochrome. They are ideal for a consistent look.
- **Original (coloured)**: These icons retain their original colors, which can make them more visually distinct but may not blend as well with your system theme.

Here's an example of how the two styles look, with monochrome icons on the top and coloured icons on the bottom:

![Icon style comparison](/styles.png)

### Icon Visibility
You can choose which icons to show in the panel and which to hide in the menu. This allows you to keep important icons visible while hiding less frequently used ones to reduce clutter.

![Icon visibility settings](/hide_icon.png)
